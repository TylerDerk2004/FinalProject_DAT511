# NOAA Storm Events Analysis (2025)

Analysis of the U.S. NOAA Storm Database exploring the health and economic impacts of severe weather events across the United States in 2025.

## 📊 Live Report
[View on RPubs](https://rpubs.com/TylerDerkovitz/1428218) 

## Questions Explored
1. Which event types are most harmful to population health?
2. Which event types occur most in which states?
3. Which event types are associated with which months?
4. Which states incur the greatest economic damage from storms?
